# Subscription List Container

> The Subscription List Container gets subscriptions and renders it as an <LinkTo kind="Components/SubscriptionList" story="Default">SubscriptionList component</LinkTo>.

## Override `SubscriptionList`

To replace the `SubscriptionList` component that is being rendered by a custom component, one can override it like this:

<CodeOrSourceMdx>
```typescript
import {WebsiteBuilderProvider, BuilderSubscriptionListProps} from '@wepublish/website/builder'
import {SubscriptionListContainer} from '@wepublish/membership/website'

const MySubscriptionList = (props: BuilderSubscriptionListProps) => <div>My Subscription List</div>

export default function () {
  return (
    <WebsiteBuilderProvider SubscriptionList={MySubscriptionList}>
      <SubscriptionListContainer />
    </WebsiteBuilderProvider>
  )
}
```
</CodeOrSourceMdx>
