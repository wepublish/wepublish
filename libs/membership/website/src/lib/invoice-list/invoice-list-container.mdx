# Invoice List Container

> The Invoice List Container gets invoices and renders it as an <LinkTo kind="Components/InvoiceList" story="Default">InvoiceList component</LinkTo>.

## Override `InvoiceList`

To replace the `InvoiceList` component that is being rendered by a custom component, one can override it like this:

<CodeOrSourceMdx>
```typescript
import {WebsiteBuilderProvider, BuilderInvoiceListProps} from '@wepublish/website/builder'
import {InvoiceListContainer} from '@wepublish/membership/website'

const MyInvoiceList = (props: BuilderInvoiceListProps) => <div>My Invoice List</div>

export default function () {
  return (
    <WebsiteBuilderProvider InvoiceList={MyInvoiceList}>
      <InvoiceListContainer />
    </WebsiteBuilderProvider>
  )
}
```
</CodeOrSourceMdx>
