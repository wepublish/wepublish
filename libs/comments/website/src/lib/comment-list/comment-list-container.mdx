# Comment List Container

> The Comment List Container gets comments and renders it as a <LinkTo kind="Components/Comment List" story="Default">CommentList component</LinkTo>.

## Override `CommentList`

To replace the `CommentList` component that is being rendered by a custom component, one can override it like this:

<CodeOrSourceMdx>
```typescript
import {WebsiteBuilderProvider, BuilderCommentListProps, CommentListContainer} from '@wepublish/website'

const MyCommentList = (props: BuilderCommentListProps) => <div>My Comment List</div>

export default function () {
  return (
    <WebsiteBuilderProvider CommentList={MyCommentList}>
      <CommentListContainer />
    </WebsiteBuilderProvider>
  )
}
```
</CodeOrSourceMdx>
