# ui

This library was generated with [Nx](https://nx.dev).

## Re-usable ui components
### Single View
This is a common view to display a single data model. That might be a user, a subscription etc.
The usage is
```html
<SingleView>
    <SingleViewTitle/>
    <SingleViewContent>
        Your content...
    </SingleViewConten>
</SingleView>
```
