# Event List Container

> The Event List Container gets events and renders it as an <LinkTo kind="Components/EventList" story="Default">EventList component</LinkTo>.

## Override `EventList`

To replace the `EventList` component that is being rendered by a custom component, one can override it like this:

<CodeOrSourceMdx>
```typescript
import {WebsiteBuilderProvider, BuilderEventListProps, EventListContainer} from '@wepublish/website'

const MyEventList = (props: BuilderEventListProps) => <div>My Event List</div>

export default function () {
  return (
    <WebsiteBuilderProvider EventList={MyEventList}>
      <EventListContainer />
    </WebsiteBuilderProvider>
  )
}
```
</CodeOrSourceMdx>
